hexo中mathjax下划线转义的问题

发表于 2017-05-24

使用{% raw %}{% endraw %}` 避免下划线反斜杠的话，可以试试 Hexo 自带的 Tag Plugins。`{% raw %}{% endraw %}内的内容不会被渲染。在公式中的“_”两侧分别加上{% raw %}`和`{% endraw %}

缺点：虽然有效，不过改变了Markdown文件的通用性，影响向其他平台的移植。
